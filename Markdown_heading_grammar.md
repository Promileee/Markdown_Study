# Markdown标题语法
要创建标题，请在单词或短语前面加井号(`#`)。`#`的数量代表了标题的级别。例如，添加三个`#`表示创建一个三级标题(`<h3>`)
## 例子
```Markdown
# Heading level 1
```
# Heading level 1
```Markdown
## Heading level 2
```
## Heading level 2
```Markdown
### Heading level 3
```
### Heading level 3
```Markdown
#### Heading level 4
```
#### Heading level 4
```Markdown
##### Heading level 5
```
##### Heading level 5
```Markdown
###### Heading level 6
```
###### Heading level 6
# 可选语法
可以在文本下方添加任意数量的`==`号来标识一级标题，或者`--`号来标识二级标题。
## 例子
```Markdown
Heading level 1
====
```
Heading level 1
=====
```Markdown
Heading level 2
----
```
Heading level 2
----
